# Course Plan — Processes (PCB + fork)

> Week 3 | by Revolvix

## Objectives
1. Define the concept of process and the difference from a program
2. Describe the PCB structure and process states
3. Explain the operation of fork(), exec(), wait()
4. Analyse the parent-child relationship and zombie/orphan processes

## Structure (100 min)
- **Part I (35 min)**: What is a process, PCB, states
- **Part II (40 min)**: fork(), exec(), wait(), exit()
- **Part III (25 min)**: Zombie, orphans, process hierarchy

## Assessment
- Formative quiz: 12 MCQ questions
- Laboratory: fork+exec implementation in C
