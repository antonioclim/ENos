# Course Plan — Basic OS Concepts

> Operating Systems | ASE Bucharest - CSIE | 2025-2026  
> Week 2 | by Revolvix

---

## Learning Objectives

At the end of this course, the student will be able to:

1. **Explain** the mechanism of hardware and software interrupts
2. **Differentiate** between polling and interrupt-driven I/O
3. **Describe** DMA operation and its advantages
4. **Analyse** the interrupt handling flow

---

## Course Structure (100 minutes)

### Part I: Interrupts (40 min)
| Time | Topic | Method |
|------|-------|--------|
| 0-15 | What are interrupts? Hardware vs Software | Diagrams |
| 15-25 | Interrupt Vector Table (IVT) | x86 Example |
| 25-40 | Handling flow: save context → handler → restore | Animation |

### Part II: I/O and DMA (35 min)
| Time | Topic | Method |
|------|-------|--------|
| 40-55 | Polling vs Interrupts: trade-offs | Comparison |
| 55-70 | DMA: transfer without CPU | Conceptual demo |
| 70-75 | Synchronous vs asynchronous I/O | Diagrams |

### Part III: Integration (25 min)
| Time | Topic | Method |
|------|-------|--------|
| 75-90 | Interrupt priorities, nested interrupts | Scenarios |
| 90-100 | Recap and Q&A | Discussion |
