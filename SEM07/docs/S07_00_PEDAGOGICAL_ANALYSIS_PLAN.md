# S07 Pedagogical Analysis Plan — Final Evaluation

## Overview

| Aspect | Details |
|--------|---------|
| **Seminar** | S07 — Final Evaluation |
| **Duration** | Variable (evaluation period) |
| **Type** | Summative Assessment |
| **Components** | Homework (25%), Project (50%), Test (25%) |

## Learning Objectives Integration

This evaluation session assesses ALL learning objectives from SEM01-SEM06:

### SEM01: Shell Fundamentals
- LO1.1-LO1.14: Basic shell operations, navigation, variables

### SEM02: I/O Redirection & Control Flow
- LO2.1-LO2.12: Pipes, redirections, loops, conditionals

### SEM03: Advanced File Operations
- LO3.1-LO3.10: find, xargs, permissions, automation

### SEM04: Text Processing
- LO4.1-LO4.12: grep, sed, awk, regular expressions

### SEM05: Functions & Data Structures
- LO5.1-LO5.10: Functions, arrays, error handling

### SEM06: Capstone Integration
- LO6.1-LO6.8: System integration, deployment, monitoring

## Assessment Philosophy

1. **Authenticity** — All assessments test real-world applicable skills
2. **Fairness** — Clear rubrics, consistent application
3. **Anti-plagiarism** — Oral verification, randomisation, AI detection
4. **Feedback** — Detailed feedback for learning improvement

## Bloom Taxonomy Distribution (Cumulative)

| Level | Target | Assessment Method |
|-------|--------|-------------------|
| Remember | 10-15% | Multiple choice, definitions |
| Understand | 20-25% | Explain, compare questions |
| Apply | 25-30% | Practical exercises, coding |
| Analyse | 15-20% | Debugging, log analysis |
| Evaluate | 5-10% | Code review, design critique |
| Create | 5-10% | Project, original solutions |
