## 1. Interactive Testing Resources

### regex101.com
The most useful tool for learning and testing regular expressions.

**Features:**
- Real-time visualisation of matches
- Detailed explanations for each part of the pattern
- Support for multiple dialects (PCRE, Python, JavaScript, Go)
- Save and share patterns
- Integrated quick reference

**Recommendation:** Test ANY regex here before using it in scripts.

**Link:** https://regex101.com

### explainshell.com
Explains shell commands in detail, argument by argument.

**Usage:** Paste a complex command and get an explanation for each part.

**Link:** https://explainshell.com

### ShellCheck Online
Online Bash script verifier.

**Link:** https://www.shellcheck.net

