Recording

üî¥
RECORDING IN PROGRESS
After you start, all your commands are being recorded!

How to stop the recording:

STOP_homework

or press Ctrl+D

Tips for a good recording:

  • Type clearly - do not rush
  • Comment what you are doing (optional)
  • Verify results before stopping

What NOT to do:

  • DO NOT delete mistakes - they show you are learning from them
  • DO NOT use clear excessively

"Connection refused" during upload

‚ñº

Possible causes:

  • You are not connected to the internet
  • The server is temporarily unavailable
  • You are on a restricted network

Solution: The file is saved locally. Try again later.

‚ñº

# Must be Python 3.8 or newer

‚ñº

Run the script again and enter the correct data. The previous file will be overwritten.

About the Cryptographic Signature

Each recording is digitally signed with RSA. This guarantees:

  • ‚úÖ Authenticity - the instructor verifies that you created the file
  • ‚úÖ Integrity - the file cannot be modified after signing
  • ‚úÖ Non-repudiation - you cannot deny that you sent the homework
üîí
You CANNOT forge another student's signature!
