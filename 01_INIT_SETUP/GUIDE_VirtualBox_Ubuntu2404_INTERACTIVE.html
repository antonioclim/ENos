When should you use this guide?
This guide is an alternative to WSL2. Use it if:
  • You have Mac (macOS) or Linux instead of Windows
  • You cannot install WSL2 on Windows
  • You prefer a complete virtual machine

What will you have at the end?

A complete Ubuntu Linux server running in a window on your computer.

How long does it take?

Approximately 60-90 minutes.

14. Common problems

‚ùå VirtualBox does not start - virtualisation error ‚ñº

You need to enable virtualisation in BIOS:

  1. Restart the computer
  2. Press Del, F2, F10 or F12 at boot
  3. Look for "Virtualization", "VT-x", "AMD-V" or "SVM"
  4. Change to "Enabled"
  5. Save and exit
‚ùå Ubuntu does not receive IP ‚ñº

Check Bridge settings in VirtualBox. Make sure you selected the correct interface (the one connected to internet).

In Ubuntu, try:

‚¨õ UBUNTU VM
sudo dhclient -v enp0s3
‚ùå Cannot connect via SSH ‚ñº
  1. Is the Ubuntu VM running?
  2. Is SSH running? sudo systemctl status ssh
  3. Is the IP correct? hostname -I
  4. Can you ping from host to VM?
‚ùå I forgot the password ‚ñº
  1. At boot, hold Shift for the GRUB menu
  2. Select "Advanced options" ‚Üí "(recovery mode)"
  3. Select "root - Drop to root shell"
  4. Type: passwd yourname
  5. Enter the new password twice
  6. Type: reboot
