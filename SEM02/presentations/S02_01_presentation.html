stderr Redirection

# stderr separately

# stdout and stderr to different files

# Combine stdout and stderr

# Short syntax (bash 4+)

Order matters!

# CORRECT: first stdout, then 2>&1
cmd > file 2>&1

# WRONG: 2>&1 points to terminal (original stdout)

Input Redirection

Loops in Bash

for while until break continue

The for Loop - List

# Basic syntax
for variable in list; do
    commands
done

# Example with explicit list
for colour in red green blue; do
    echo "Colour: $colour"
done

# Example with globbing
for file in *.txt; do
    echo "Processing: $file"
    wc -l "$file"
done

The for Loop - Brace Expansion

# Sequence of numbers
for i in {1..5}; do
    echo "Number: $i"
done

# With step
for i in {0..10..2}; do
    echo $i  # 0, 2, 4, 6, 8, 10
done

# Letters
for letter in {a..z}; do
    echo -n "$letter "
done

IMPORTANT PITFALL!

# Does NOT work with variables!
N=5
for i in {1..$N}; do echo $i; done
# Output: {1..5} (literal!)

Brace expansion happens at parse time, before variable expansion!

Solutions for {1..$N}

Method 1: seq

N=5
for i in $(seq 1 $N); do
    echo $i
done

Method 2: C-style

N=5
for ((i=1; i<=N; i++)); do
    echo $i
done

Method 3: while

N=5; i=1
while [ $i -le $N ]; do
    echo $i
    ((i++))
done

The while Loop

# Syntax
while condition; do
    commands
done

# Example: countdown
count=5
while [ $count -gt 0 ]; do
    echo "T minus $count"
    ((count--))
    sleep 1
done
echo "LAUNCH!"

# Infinite loop (useful for daemons)
while true; do
    echo "Running..."
    sleep 60
done

Pattern: while read

Read a file line by line

# CORRECT: with redirection
while IFS= read -r line; do
    echo "Line: $line"
done < input.txt

# IFS= preserves leading/trailing spaces
# -r disables backslash interpretation

PITFALL: Pipe creates SUBSHELL!

# WRONG - variables do NOT persist!
count=0
cat file.txt | while read line; do
    ((count++))
done
echo $count  # 0 - subshell problem!

The until Loop

Opposite of while: executes UNTIL the condition becomes true

# Wait until the file exists
until [ -f /tmp/ready.flag ]; do
    echo "Waiting..."
    sleep 5
done
echo "File is ready!"

# Equivalent with while NOT
while [ ! -f /tmp/ready.flag ]; do
    echo "Waiting..."
    sleep 5
done

ğŸ’¡ Useful for wait-for-condition patterns

â¹ï¸ Control Flow: break and continue

break

Exits the current loop

for i in {1..10}; do
    if [ $i -eq 5 ]; then
        break
    fi
    echo $i
done
# Output: 1 2 3 4

continue

Skips to the next iteration

for i in {1..5}; do
    if [ $i -eq 3 ]; then
        continue
    fi
    echo $i
done
# Output: 1 2 4 5

ğŸ’¡ break N and continue N affect N levels of loops
