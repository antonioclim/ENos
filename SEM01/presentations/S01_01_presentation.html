Wildcards

File Globbing (Pattern Matching)

Pattern Description Example Matches
* Zero or more characters *.txt file.txt, note.txt, a.txt
? Exactly one character file?.txt file1.txt, fileA.txt (NU file10.txt!)
[abc] One character from set file[123].txt file1.txt, file2.txt, file3.txt
[a-z] One character from range file[a-c].txt filea.txt, fileb.txt, filec.txt
{a,b,c} Brace expansion (lists) file{1,2,3}.txt file1.txt file2.txt file3.txt
âš ï¸
Trap: * does NOT include hidden files (those starting with . )! Use .* or ls -a.
