  • pwd Print Working Directory - displays current directoryy

  • cd ~ Go to HOME directoryy

üìÅ Files & Directories
  • mkdir -p DIR Create directoryy(ies) recursively
  • touch FILE Create empty file or update timestamp
  • cp SRC DEST Copy files (-r for directoryies)
  • mv SRC DEST Move/Rename
  • rm FILE Delete file (-rf for directoryies)
  • cat FILE Display file contents
‚ö†Ô∏è Trap: rm -rf deletes permanently! There is no Recycle Bin.
üì¶ Variables

  • unset VAR Delete the variable
  • env Display all environment variables
